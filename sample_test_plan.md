# Sample Test Plan — Space Miner (demo)

Scope
-----
This plan covers functional and exploratory testing of the Space Miner prototype (mobile). Focus is on core gameplay loops, player controls, item collection, level progression, and stability.

Test types
----------
- Functional test cases
- Exploratory sessions
- Regression checks after bug fixes
- Compatibility on Android emulators and a physical low-end device

Example test cases
------------------
1. Launch game — verify main menu loads and buttons respond
2. Start level 1 — verify player movement, collisions, and item collection
3. Resource pickup — confirm inventory increments and no physics clipping
4. Pause/resume — ensure game state preserved

Reporting
---------
Use a bug template with clear steps, expected vs observed behavior, severity, device details, and logs/screenshots when possible.
